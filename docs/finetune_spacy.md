# Дообучение spaCy NER

## Цель

По итогам сравнения моделей (см. `experiments.md`) была поставлена задача обучить компактную NER-модель на два класса — **NAME** и **ADDRESS** — достаточно точную для встраивания в прокси-сервис и достаточно быструю для инференса в реальном времени.

## Базовая модель

Дообучалась модель **`ru_core_news_lg`** (spaCy, русский язык). Из неё был взят предобученный компонент `tok2vec` (векторные представления слов), остальные компоненты (парсер, морфологизатор и пр.) отброшены. Поверх `tok2vec` обучен NER-head с нуля на двух метках: `NAME` и `ADDRESS`.

## Обучающие данные

Использовалось три источника данных.

### 1. scanpatch/pii-ner-corpus-synthetic-controlled

Синтетический датасет с HuggingFace. Содержит 26 типов ПД-сущностей (включая гранулярные компоненты имени и адреса: `first_name`, `last_name`, `address_city`, `address_street` и т.д.). При подготовке все компоненты имени объединены в `NAME`, все компоненты адреса — в `ADDRESS`. Тексты отфильтрованы по кириллице.

Предварительно по алфавиты были отфильтрованы примеры только на русском.

### 2. AlexKly/Detailed-NER-Dataset-RU

Датасет в BIOLU-разметке для русского NER. Содержит гранулярные метки (`FIRST_NAME`, `LAST_NAME`, `CITY`, `STREET`, `HOUSE` и др.). При подготовке смежные токены, относящиеся к одному адресу или имени, объединялись в единую сущность по допустимым парам (город+улица, улица+дом и т.д.).

### 3. Синтетический датасет (собственный)

Датасет, сгенерированный в рамках этого проекта (см. `synth_generation.md`). 3 030 примеров по 13 доменам, с негативными примерами и граничными случаями.

## Конфигурации обучения

Для оценки вклада каждого источника данных обучены три модели:

| Конфигурация | Обучающие данные                         | Модель                   |
|--------------|------------------------------------------|--------------------------|
| **2ds**      | scanpatch + AlexKly                      | `model-best-2ds-0.52/`   |
| **synth**    | только собственный синтетический датасет | `model-best-synth-0.63/` |
| **combo**    | scanpatch + AlexKly + синтетический      | `model-best-combo-0.78/` |

## Результаты

Оценка на `hivetrace/pii-bench` (exact-match).
